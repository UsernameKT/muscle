# マッスルメモリー

## サイト概要
### サイトテーマ
日々の筋力トレーニングを記録するSNSサイト
​
### テーマを選んだ理由
筋力トレーニングをする際に目標を見失ってしまったり、達成した実感がないとなかなか継続できないという悩みがありました。
私は手書きでノートに記述をして過去の記録を見返していましたが、ノートはかさばり書くのは面倒。また、無くしてしまったり水に濡らしてしまうと記録を失ってしまうリスクもあります。
そのためサイトによって一括で管理できたら手軽に記録できるし、過去を振り返ることも容易で便利だと考えました。
また、どういったトレーニングをやればよいか分からない方も多いと思います。そんな方のためにユーザーが実際に行なっているメニューを閲覧し、参考にできるようなサイトもあったら良いと思いました。
運動上級者の方だけでなく、これから始めたいと思う初心者の方でも挫折することなく楽しんで続けてほしいと思いこのテーマにしました。

​
### ターゲットユーザ
・トレーニングが好きで種目や回数などを記録していきたい人

・運動する習慣を身に付けたい人

​
### 主な利用シーン
・いつトレーニングを実施したかを確認したいとき

・過去の自分の重量や回数などを確認したいとき
